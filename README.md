### Задание 1.

Приложите ссылку на самую сильную вашу работу, собранную через вайбкодинг, и кратко опишите: что это, в каком инструменте делали и что было вашей задачей.

**Ответ:**

Ссылка: https://signdrop.site

SignDrop — инструмент для наложения рукописной подписи на документы (PDF, JPEG, PNG, TIFF, WEBP). Работает полностью локально, без облака и регистрации: открыл файл → перетащил подпись → сохранил.

Инструменты: Claude Code + Cursor (ежедневный рабочий стек).

Моя задача: вела продукт от размытой личной потребности («мне нужно чем-то подписывать документы онлайн») до полноценного готового продукта — сама, без передачи задач другим. Это включало:

- **Архитектуру и бэкенд:** FastAPI + PyMuPDF для работы с PDF, алгоритм на основе яркости пикселей для автоматического удаления фона подписи (офлайн, без нейросети — точнее и без утечки данных)
- **Фронтенд:** React + Vite + Konva.js для интерактивного канваса (drag & drop, resize, поворот подписи, живое превью)
- **Десктоп-сборку:** Tauri + PyInstaller, кросс-платформенные релизы под Windows / macOS / Linux через GitHub Actions
- **Продуктовую упаковку:** собственный лендинг с нуля (signdrop.site) — SEO, метатеги, OpenGraph, анимация, адаптивная вёрстка, интеграция донатов
