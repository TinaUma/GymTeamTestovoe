# Тестовое задание — Вайбкодер / Сборщик лендингов на AI (Усманова)

---

## Задание 1 — самая сильная работа

**Ссылка:** https://signdrop.site

**SignDrop** — инструмент для наложения рукописной подписи на документы (PDF, JPEG, PNG, TIFF, WEBP). Работает полностью локально, без облака и регистрации: открыл файл → перетащил подпись → сохранил.

**Инструменты:** Claude Code + Cursor (ежедневный рабочий стек)

**Моя задача:** вела продукт от размытой личной потребности до полноценного готового продукта — сама, без передачи задач другим:

- **Архитектура и бэкенд:** FastAPI + PyMuPDF для работы с PDF, алгоритм на основе яркости пикселей для автоматического удаления фона подписи (офлайн, без нейросети — точнее и без утечки данных)
- **Фронтенд:** React + Vite + Konva.js для интерактивного канваса (drag & drop, resize, поворот подписи, живое превью)
- **Десктоп-сборка:** Tauri + PyInstaller, кросс-платформенные релизы под Windows / macOS / Linux через GitHub Actions
- **Продуктовая упаковка:** лендинг с нуля (signdrop.site) — SEO, метатеги, OpenGraph, анимация, адаптивная вёрстка, интеграция донатов

## Задание 2 — воспроизведение лендинга Усмановой

**Живая ссылка:** https://tinauma.github.io/GymTeamTestovoe/

Воспроизведены три экрана лендинга [usmanovafit.gymteam.ru/mainpage](https://usmanovafit.gymteam.ru/mainpage):

2. **О Кате** — буллеты с достижениями, галерея фото
3. **Тренировки дома** — карточки программ с фото, тегами и кнопками

**Инструменты:** Claude Design (дизайн и вёрстка) + Claude Code (деплой на GitHub Pages)
